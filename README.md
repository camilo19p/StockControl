# StockControl

Sistema de control de inventario para microempresas de licores en Cartagena.

## Descripción
StockControl es una aplicación de escritorio desarrollada en **Java** con interfaz gráfica (Swing/NetBeans) que optimiza la gestión de inventario. Reemplaza los registros manuales permitiendo:

- Control de entradas y salidas de productos
- Registro de ventas y ganancias totales
- Generación de reportes
- Búsqueda, actualización y eliminación de productos

## Tecnologías utilizadas
- Java (JDK 8 o superior)
- Swing para la interfaz gráfica
- NetBeans como IDE
